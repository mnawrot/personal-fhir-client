Forwarding to your local Personal FHIR Client…

This page is a static redirect shim. Some health-plan and EHR developer portals (e.g. BCBS, Cerner) require a public HTTPS redirect URI and won't accept the 127.0.0.1 loopback that RFC 8252 recommends for native apps. This page forwards the OAuth authorization code from that public URL back to your local app listener at https://127.0.0.1:8765/smart/callback.
