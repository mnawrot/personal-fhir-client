Personal FHIR Client

A patient-facing SMART on FHIR client. Lets an individual patient pull their own structured medical records from Epic-powered patient portals and store them locally as JSON, for longitudinal self-tracking and personal health analytics.

Source & documentation

Code, issues, and detailed README: github.com/mnawrot/personal-fhir-client

Terms of use: TERMS.md

What this page is for

This domain (mnawrot.github.io/personal-fhir-client/) hosts two static files used by the client:

  • This landing page (what you're reading).
  • An OAuth redirect shim at /callback.html, used to forward an OAuth authorization code from providers that require a public HTTPS redirect URI (e.g., BCBS, Cerner) back to the local app listener on https://127.0.0.1:8765/.

Privacy

This page and the callback shim contain no tracking and store no data. The OAuth code is in the URL only during the redirect and is relayed to your local machine only. See the full Terms of use for details.
